⚙️ ตั้งค่าการเชื่อมต่อ Google Sheets

ระบบดึงข้อมูลจาก Google Sheets ผ่าน Google Apps Script — ทำครั้งเดียว ใช้ได้ทุกเครื่อง

1
เปิด Google Sheets → ส่วนขยาย (Extensions) → Apps Script → ลบโค้ดเดิมออกทั้งหมด แล้ว วางโค้ดใหม่
2
Deploy → New Deployment · Type: Web app · Execute as: Me · Who has access: Anyone → Deploy → Authorize → Allow
3
เปิด URL ที่ได้ตามด้วย ?action=setup เพื่อสร้างระบบ User ครั้งแรก
4
Copy URL (ไม่ต้องมี ?action=setup) → วางด้านล่าง → กด เชื่อมต่อ
🔗 URL ของ Apps Script
🔗 ลิงค์แชร์ให้ทีม
